#  AI Resume Analyzer

An AI-powered web app that analyzes resumes against job descriptions and provides:

-  Match Score  
-  Skill Breakdown  
-  Improvement Suggestions  

---

##  Tech Stack
- HTML, CSS, JavaScript  
- Node.js + Express  
- OpenAI API (with fallback system)  
- PDF.js  

---

##  Features
- Upload PDF resume  
- Compare with job description  
- Visual score + skill analysis  
- Works even without API credits (fallback system)
